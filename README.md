# Particles
Particle System - The physics kind!

## Planned Mechanics
* Add particles
* Simulate:
  * Charge
  * Velocity
  * Collision
  * Radioactivity

## Controls
* +/-: Edit charge
* [/]: Edit mass
* ;/': Edit radioactivity
* Left click: Create particle
* Left click, drag, release: Launch particle
* Space: Pause
* Backspace: Clear simulation
